Тест №15

Инструкция

  • 1.Выберите один из вариантов в каждом из 5 вопросов;
  • 2.Нажмите на кнопку "Показать результат";
  • 3.Скрипт не покажет результат, пока Вы не ответите на все вопросы;
  • 4.Загляните в окно рядом с номером задания. Если ответ правильный, то там (+). Если Вы ошиблись, там (-).
  • 5.За каждый правильный ответ начисляется 1 балл;
  • 6.Оценки: менее 3 баллов - НЕУДОВЛЕТВОРИТЕЛЬНО, от 3 но менее 4 - УДОВЛЕТВОРИТЕЛЬНО, 4 и менее 5 - ХОРОШО, 5 - ОТЛИЧНО;
  • 7.Чтобы сбросить результат тестирования, нажать кнопку "Сбросить ответы";
  1. Первым этапом создания микросхемы является:
    1. Формирование фоторезистора.
    2. Формирование слоя диоксида кремния.
    3. Фотолиграфия
    4. Травление.

  2. Шестым этапом создания микросхем является:
    1. формирование фоторезистора.
    2. травление.
    3. формирование слоя диоксида кремния.
    4. химическая обработка для удаления остатков фоторезиста.

  3. Вторым этапом создания микросхем является :
    1. химическая обработка для удаления остатков фоторезиста.
    2. формирование слоя диоксида кремния..
    3. формирование фоторезистора.
    4. травление.

  4. Третим этапом создания микросхем является :
    1. фотолиграфия.
    2. химическая обработка для удаления остатков фоторезиста.
    3. травление.
    4. формирование фоторезистора.

  5. Пятым этапом создания микросхем является :
    1. травление.
    2. формирование фоторезистора.
    3. химическая обработка для удаления остатков фоторезиста.
    4. фотолиграфия.
