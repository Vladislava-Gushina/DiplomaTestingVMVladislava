Порядок выполнения работы:

 Состав системы P-CAD 2001

Система P-CAD 2001 представляет собой интегрированный пакет программ, предназначенный для проектирования многослойных печатных плат (ПП) радиоэлектронных средств (РЭС) [1,2]. Она адаптирована к операционной среде Windows и использует все настройки и возможности последней. P-CAD 2001 включает в себя следующие программные модули: P-CAD Library Executive, P-CAD Schematic, P-CAD PCB, P-CAD Autorouters, Symbol Editor, Pattern Editor, InterPlace PCS, Relay, Signal Integrity.

Утилита Library Executive (Администратор библиотек) состоит из программы Library Manager (Менеджер библиотек), редактора символов элементов Symbol Editor и редактора посадочных мест Pattern Editor электрорадиоэлементов (ЭРЭ) на ПП. P-CAD Schematic — графический редактор электрических схем. Он предназначен для разработки электрических принципиальных схем и может применяться для создания условных графических обозначений (УГО) отдельных ЭРЭ (файлы с расширением .sch). P-CAD PCB — графический редактор ПП. Предназначен для проектирования конструкторско-технологических параметров ПП. К ним относятся: задание размеров ПП, ширина проводников, величина зазоров, размер контактных площадок, диаметр переходных отверстий (ПО), задание экранных слоев, маркировка, размещения ЭРЭ, неавтоматическая трассировка проводников и формирование управляющих файлов технологическим оборудованием. P-CAD Autorouters предназначен для автоматической трассировки проводников ПП.

Включает два автотрассировщика: программу Quick Route для проектирования рисунка ПП не очень i сложных электрических схем и бессеточный трассировщик Shape-Based Router, предназначенный для проектирования многослойных ПИ с высокой плотностью расположения ЭРЭ. Symbol Editor — редактор символов элементов (файлы с расширением .sym). Предназначен для создания условных графических обозначений символов ЭРЭ электрических схем. Pattern Editor — редактор посадочных мест (файлы с расширением .pat). Предназначен для разработки посадочных мест для конструктивных ЭРЭ на ПП. InterPlace PCS — программа интерактивного размещения ЭРЭ. Relay — программа просмотра ПП, расстановки ЭРЭ на ней, задания основных атрибутов, контроля технологических ограничений. Signal Integrity — программа анализа электрических параметров ПП. SPECCTRA — программа размещения ЭРЭ на ПП, ручной, интерактивной и автоматической трассировки проводников. Благодаря бессеточной технологии является очень эффективным трассировщиком ПП высокой степени сложности и с высокой плотностью размещения ЭРЭ. Информация о ПП в SPECCTRA передается через редактор РСВ.

7.2. Основные технические характеристики системы

Система P-CAD 2001 устанавливается на компьютер с процессором 486 и Pentium. Работает под управлением Microsoft Windows 95/98/2000/NT. На жестком диске она вместе с библиотеками и утилитами занимает около 350 Мб. Минимальный объем ОЗУ составляет 16 Мб, но рекомендуется 32 Мб, минимальное разрешение монитора 800x600. Система P-CAD 2001 позволяет выполнять проекты:  на печатных платах, не превышающих размер 60x60 дюймов;  с шириной трассы не более 1 см (394 мил);  с минимальным шагом сетки 0,1 мил в английской системе и 0,001 мм = 1 мкм в метрической системе (1 мил = 0,001 дюйма = 0,0254 мм). Включающие:  не более 99 листов схем в проекте с максимальным размером листа 60x60 дюймов;  не более 64 000 цепей в проекте;  не более 999 выводов в компоненте;

не более 255 секций в компоненте;  не более 20 000 компонентов в одной библиотеке;  не болеее 99 слоев в базе данных ПП (включая сигнальные слои, слои металлизации и вспомогательные);  не более 64 000 типов контактных площадок в проекте;  не более 64 000 стилей стеков контактных площадок в проекте.

7.3. Запуск системы

Запуск программ системы P-CAD 2001 выполняется следующим образом. Щелчком ЛК по кнопке «Пуск» в выпадающем меню команд «Программы» найти «P-CAD 2001» и щелкнуть ЛК по ней. В открывшемся меню ряд названий будет начинаться с P-CAD. Это и есть программные модули P-CAD 2001. Для запуска любого из них достаточно щелкнуть по требуемому, и программа откроется. В том случае, если на компьютере запущена одна из программ P-CAD 2001, необходимо ЛК мыши щелкнуть по команде Utils (Служебные команды) во второй строке сверху. Щелчок мыши по Symbol Editor запустит программу.При этом действующая программа не закроется, а только свернется, и к ней всегда можно будет вернуться. Экран графического редактора P-CAD Symbol  Editor представлен на рисунок 8.1.1.

10.1. Основные сведения о программе P-CAD Pattern Editor

Графический редактор P-CAD Pattern Editor имеет набор команд, позволяющих создавать и редактировать посадочные места для установки ЭРЭ на печатных платах. Программа работает с файлами отдельных посадочных мест (.pat) и библиотек (.lib). Посадочное место (ПМ) — это комплект конструктивных элементов печатной платы, предназначенный для монтажа отдельного ЭРЭ. В него входят в различных сочетаниях контактные площадки (КП), металлизированные отверстия, печатные проводники на наружных слоях и гладкие крепежные отверстия. Запуск программы осуществляется через кнопку «Пуск» с последующим выполнением в выпадающем меню команд «Программы», «PCAD 2001» и «Pattern Editor». Пояснение к работе:

11.1. Основные сведения о программе P-CAD Pattern Editor

Графический редактор P-CAD Pattern Editor имеет набор команд, позволяющих создавать и редактировать посадочные места для установки ЭРЭ на печатных платах. Программа работает с файлами отдельных посадочных мест (.pat) и библиотек (.lib). Посадочное место (ПМ) — это комплект конструктивных элементов печатной платы, предназначенный для монтажа отдельного ЭРЭ. В него входят в различных сочетаниях контактные площадки (КП), металлизированные отверстия, печатные проводники на наружных слоях и гладкие крепежные отверстия. Кроме этого ПМ может включать в себя параметры защитной и паяльной масок, элементы маркировки и графические элементы сборочного чертежа.

Запуск программы осуществляется через кнопку «Пуск» с последующим выполнением в выпадающем меню команд «Программы», «PCAD 2001» и «Pattern Editor». В том случае, если на компьютере запущена одна из программ P-CAD 2001, необходимо щелкнуть ЛК по команде Utils (Служебные команды). Откроется выпадающее меню, в котором несколько пунктов начинаются с аббревиатуры P-CAD. Щелчок мыши по P-CAD Pattern Editor запустит программу. При этом действующая программа не закроется, а только свернется и к ней всегда можно будет вернуться. Экран графического редактора P-CAD Pattern Editor представлен на рисунок 11.1.1.

14.1. Общие сведения о графическом редакторе P-CAD Schematic

Графический редактор P-CAD Schematic предназначен для разработки электрических принципиальных схем с использованием условных графических обозначений элементов. При этом УГО ЭРЭ могут извлекаться из соответствующей библиотеки или создаваться средствами самой  программы.

Если не разрабатывается узел печатной платы, то при вычерчивании схем берутся УГО элементов, не связанные с их конструктивной базой.

Такая схема может использоваться как иллюстративный материал. При возникновении необходимости разработки ПП ее надо дополнить соответствующей конструкторско-технологической информацией. При выполнении проекта с разработкой узла ПП схема должна формироваться из библиотечных элементов, которые включают полную информацию о конструктивных особенностях ЭРЭ и их посадочных местах на ПП.

Запуск программы осуществляется через кнопку «Пуск» с последующим выполнением в выпадающем меню команд «Программы», «PCAD 2001» и «P-CAD Schematic». В том случае, если на компьютере запущена одна из программ P-CAD 2001, необходимо щелкнуть ЛК по команде Utils (Служебные команды).

Откроется выпадающее меню, в котором несколько пунктов начинаются с аббревиатуры P-CAD. Щелчок мыши по P-CAD Schematic запустит программу. При этом действующая программа не закроется, а только свернется и к ней всегда можно будет вернуться. Экран графического редактора P-CAD Schematic представлен на рисунок 14.1.1.

Цель работы:  экспортировать файлы, полученные в P-CAD в отечественную программу Компас для выпуска конструкторско-чертежной документации согласно российским стандартам

Требование по охране труда:

При выполнении лабораторных работ соблюдать требования действующих Положений по охране труда, инструкция №1.

Приборы и элементы: ПО P-CAD, ПК.

Пояснение к работе: В этой работе мы научимся конвертировать файл

Задание: Изучить методические рекомендации по проведению лабораторной работы.

Порядок выполнения работы:

Данное преобразование позволяет конвертировать файлы с расширением .sch и  .pcb в новый формат с расширением .dxf, который может быть использован при работе с другими программами проектирования печатных плат и рисования схем, а также в AutoCAD различных версий, в Компас.

  Приступая к преобразованию, следует помнить, что в полученном формате будет сохранена вся информация исходного проекта, отображенная на видовом экране P-CAD , но воспроизводится она будет без разделения по слоям. То есть все, что ранее изображалось на нескольких слоях, в новом формате получится на одном. Поэтому, прежде чем преобразовывать любой файл, его следует соответственным образом подготовить.

   Преобразование файлов электрических схем, выполненных на нескольких листах, но размещенным на одном видовом экране (фактически на одном большом листе), необходимо производить последовательно, лист за листом. Таким образом, для схемы, разработанной на нескольких листах, будет получен ряд самостоятельных фалов в формате DXF.

  Конструкторско–технологические проекты (файлы с расширением .pcb) тоже надо конвертировать отдельными слоями или их группами. В этом случае необходимо определить, что должно войти в состав нового файла. Например, можно сгруппировать всю информацию,  относящуюся к первому (верхнему) слою, и преобразовать ее в виде самостоятельного файла формата DXF, а затем подобрать то, что относится к последнему слою,  и создать второй файл. Здесь следует исходить из того, что после преобразования вся информация отобразится на одном слое, поэтому разделение должно быть выполнено заранее. 

  После подготовки очередного видового экрана в программе P-CAD (должны быть включены одни и  выключены другие слои ) следует создать промежуточный файл с расширением .plt. Для этого включите рамкой управления команду SYS, а затем PLOT, после чего, работая указателем мыши в режиме рамки, выделите ту часть выделенного на экран проекта, которую планируется поместить в новый файл. Это может быть как весь проект, так и любой его фрагмент.

  Далее обратите внимание на командную строку в нижней части экрана с надписью Plot file name: <имя файла> .plt,  которая повторяет имя исходного файла. В случае создания нескольких файлов для разных составных частей проекта (например, отдельных листов схемы) вам понадобится вводить (изменять) название каждого файла из группы (пачки). В такой ситуации проще всего добавлять к существующему имени дополнительные цифры,   

например:   -1, -2 и т.д.

 Изменив ( или сохранив без изменения) имя нового файла, щелкните ЛК. Выбранный проект  (или его часть ) будет преобразован и записан  с новым названием и расширением.

Таким образом  можно подготовить требуемое количество новых файлов с расширением .plt. При этом следует быть внимательными,  потому что вывести на экран и проверить содержимое этих файлов невозможно, пока они не будут вновь конвертированы в формат, читаемый одной из программ проектирования.

Затем в командной строке введите с клавиатуры текст:   CCONV_-A_< имя файла>. При наборе обязательны пробелы (в примере они показаны подчеркиванием) и знак “минус” перед буквой “A”. Имя файла можно набрать вручную или нажать Ctrl + Enter- выделенный цветом файл будет переписан в командную строку.

Этот процесс займет определенное время, по истечении которого в списке появится новый файл с расширением .dxf.  Таким образом можно поочередно преобразовывать все подготовленные файлы.
